此指南用于 Docker/1Panel 环境部署，适用于生产服务器。

一、准备环境变量

  1. 设置数据库连接：DATABASE_URL
  2. 设置管理员账号：ADMIN_USERNAME 与 ADMIN_PASSWORD
  3. 设置 JWT 密钥：JWT_SECRET
  4. 管理员面板端口：ADMIN_PORT

二、启动服务

  1. 执行 docker compose up -d --build。
  2. 访问主站端口与管理员端口，确认服务可用。

三、备份与迁移

  1. 管理员面板导出 JSON 备份。
  2. 在新服务器导入备份后即可完整恢复。
