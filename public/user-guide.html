1. 登录与注册

  1. 点击首页右上角 登录 进入登录页。
  2. 若管理员开放注册，可点击 注册，按提示填写用户名、密码、邮箱与邀请码。
  3. 若账户启用二次验证（TOTP），登录时需输入验证码。

2. 公开主页访问（未登录展示）

  1. 你的公开主页地址格式为：http(s)://域名/用户名（用户名仅支持英文和数字）。
  2. 示例：http://127.0.0.1:3000/test3。
  3. 该页面仅展示你设置为公开的图标，私密图标不会显示。

3. 常用操作

  • 🔒 模式切换：预览 / 排序 / 编辑 / 增删。
  • 排序模式可拖拽图标，支持分类与 Dock 之间互拖。
  • 添加或编辑网址时，可直接选择 https:// 或 http:// 前缀。

4. 资料与安全

  • 头像菜单 → 编辑资料：可修改用户名、头像、密码。
  • 用户名修改会自动校验是否重复。
  • 设置页可开启/关闭二次验证（TOTP）。

5. 备份与恢复

在设置页可导出/导入个人数据。导入将覆盖当前账户内容，建议操作前先导出备份。
